# Blocks and Bridges Main Website

This is the main website for [Blocks and Bridges](https://blocksandbridges.ca/). It is built using [NextJS](https://nextjs.org/) and compiled to static HTML using Webpack.

## Development

To run the site locally, clone the repo and run `npm install` to install the dependencies. Then run `npm start` to start the development server. The site will be available at [http://localhost:3000](http://localhost:3000).

## Deployment

The site is deployed to [Vercel](https://www.vercel.com/) using a GitHub App. Pushing to the `main` branch will trigger a new deployment.
